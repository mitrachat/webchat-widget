MitraChat Widget - Local Test Environment

Instructions:
  1. Make sure the backend server is running: bun run dev:server
  2. Create a webchat provider in the dashboard
  3. Copy the provider ID and paste it below
  4. Click "Load Widget" to test

Widget Configuration

Note: Make sure your provider has the current domain (localhost) in the allowed domains list, or leave it empty to allow all domains.

Embed Code Preview

<!-- Add this to your HTML --> <script src="./dist/embed.iife.js"></script> <script> window.MitraChatWidget.init({ providerId: 'YOUR_PROVIDER_ID', apiUrl: 'http://localhost:3000' }); </script>

Alternative: NPM Import

import { mountWebchat } from '@mitrachat/webchat-widget'; mountWebchat({ providerId: 'YOUR_PROVIDER_ID', apiUrl: 'http://localhost:3000', theme: 'auto', position: 'bottom-right' });
